# DBT

## Repositório de exemplo de projeto

https://github.com/Linhares015/athena_dbt

## Criar venv

```bash
python3 -m venv venv
```

## Subir venv

```bash
source venv/bin/activate
```

## Instalar o DBT via repositório de exemplo

```bash
pip install -r requirements.txt
```

O DBT é por versão que você irá usar no banco de dados no nosso exemplo usamos postgres mas depende de qual será o seu.
